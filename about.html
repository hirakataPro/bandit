About

このサイトについて

Shell は、ターミナルを初めて触る方が、コマンドを使ってサーバーを探索する感覚を身につけるための日本語学習サイトです。ブラウザの中に小さな仮想サーバー（35のレベル）を用意し、実際にコマンドを打ち込みながら答えを探していく形式で進みます。

このサイトのねらい

サーバーや Linux に初めて触れる方が、教科書だけではなかなか身につかない「自分の手でターミナルを動かす感覚」を、安全な環境で体験できるようにすることを目指しています。間違えても何も壊れません。気が済むまで試してみてください。

OverTheWire Bandit について

本サイトは、海外で長年親しまれている OverTheWire Bandit という学習用ウォーゲームに着想を得ています。各レベルが扱う「学べる技術テーマ」を参考にしていますが、本サイトに掲載されている問題文・シナリオ・解説・ヒント・サンプルは、すべて Shell が独自に新規作成したオリジナルコンテンツです。

本サイトは Bandit 原文の翻訳でも翻案でもありません。実際にネットワーク越しに本物のサーバーへ接続して挑戦してみたい方は、ぜひ原典の OverTheWire Bandit をご利用ください。本物のサーバーで挑戦すると、また違った学びがあります。

進捗の保存について

あなたの学習進捗は、ご利用のブラウザの中（localStorage）に自動的に保存されます。サーバーには何も送信しません。端末を変えるとき、ブラウザの履歴を消すときなどは、設定ページの「進捗をエクスポート」機能で JSON ファイルとしてバックアップしておくと安心です。

プライベートブラウジングなどで自動保存が無効な場合は、画面上部に通知が出ます。

免責

本サイトのコマンド実行はすべて、ブラウザの中だけで動作する仮想ターミナル上で行われます。あなたの実際のコンピュータには影響しません。あくまで学習用のシミュレーションであり、実環境の Linux と挙動が異なる部分もあります。本物の感覚は、ぜひ実機やクラウド上の Linux サーバーで身につけてください。
