# Shell — ターミナルで、Linuxを身につける。

OverTheWire Bandit に着想を得た、日本語の Linux 学習サイト。
ブラウザの中に小さな仮想サーバーを用意し、35のレベルでターミナル操作と
Linux コマンドを最初から学べる。

## ステータス（フェーズ別）

| フェーズ | 内容 | 状態 |
|---|---|:---:|
| 1 | 骨格（デザインシステム / index / about / settings / 進捗保存） | ✅ |
| 2 | チートシート（106 コマンド + 検索 + カテゴリ絞り込み + コピー） | ✅ |
| 3 | 仮想ターミナル本体（VFS / パーサー / 30+ コマンド実装） | 未着手 |
| 4 | レベル統合（level.html の本実装、パスワード検証） | 未着手 |
| 5 | 全 35 レベルの問題文・シナリオ・仮想ファイルシステム執筆 | 未着手 |
| 6 | 仕上げ（a11y / レスポンシブ / アニメーション） | 未着手 |

## ローカルで動かす

このプロジェクトはビルド不要。ただしブラウザの fetch 系 API が
`file://` で動かないため、ローカルで開くときは簡易 HTTP サーバーを使う。

### Python（標準ライブラリ）

```sh
python -m http.server 8000
# → http://localhost:8000/ にアクセス
```

### Node.js

```sh
npx serve .
```

### PowerShell（Python 同梱の場合）

```powershell
python -m http.server 8000
```

## デプロイ

純粋な静的サイトなので、以下のいずれかにそのままアップロードできる:

- **GitHub Pages**: リポジトリの Settings → Pages で公開
- **Netlify**: ドラッグ＆ドロップ、または git 連携
- **Vercel**: フレームワーク検出は不要、Output Directory はリポジトリルート

注意:
- 進捗データはユーザーのブラウザの `localStorage` に保存されるため、
  デプロイ先のドメインが変わると過去の進捗は引き継がれない。
- 端末を変える / バックアップしたいときは `設定` ページの
  「進捗をエクスポート」で JSON ファイルとしてダウンロード。
